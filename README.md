# Global Layoffs Analytics Dashboard

## Project Overview
This project analyzes global layoff trends across companies, industries, countries, and time periods using SQL and Power BI.

The goal of this project is to uncover patterns and insights related to layoffs worldwide through data cleaning, transformation, exploratory data analysis, and interactive dashboard visualization.

---

## Tools & Technologies Used
- MySQL
- Power BI
- Excel

---

## Key Features
- Data cleaning and preprocessing using SQL
- KPI metrics for layoffs analysis
- Industry-wise layoff trends
- Country-wise layoffs visualization
- Top companies affected analysis
- Interactive slicers and filters
- Executive-style Power BI dashboard

## Key Insights
- Consumer and Hardware industries experienced the highest layoffs.
- Layoffs peaked significantly during 2022–2023.
- Major tech companies contributed heavily to total layoffs.
- Global layoffs impacted 47 countries and over 1300 companies.
